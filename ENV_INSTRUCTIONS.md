# Настройка переменных окружения

## Обязательные переменные

- `BOT_TOKEN` — токен вашего Telegram-бота

## Рекомендуемые переменные

- `CHANNEL_ID` — ID канала для публикации (начинается на -100...)
- `ADMIN_ID` — ваш Telegram ID
- `DATABASE_PATH` — путь к базе данных (по умолчанию `bot.db`)

## Контакты в объявлениях

Бот **всегда** показывает контакт того, кто разместил груз/машину:
1. телефон или @username из текста объявления;
2. иначе Telegram @username автора;
3. иначе имя из профиля Telegram.

`CONTACT_USERNAME` **не подставляется** в объявления (раньше из‑за этого
везде светился один и тот же контакт владельца).

## Рекомендуемые значения

```
BOT_TOKEN=...
CHANNEL_ID=-100xxxxxxxxxx
DATABASE_PATH=bot.db
```

**Как настроить:**
1. Скопируйте `.env.example` → `.env`
2. Заполните значения в `.env`
3. Добавьте переменные в Railway (Variables)
4. **Никогда** не коммитьте `.env` в git
